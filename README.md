# NotElec

Electron-App for a famous Organizing Tool

# Requirements
- node.js
- npm
- Electron^6

# Installation

- Navigate to the 'NotionElectron-master' folder
	- run `chmod +x Install.sh && ./Install.sh`

# Uninstallation

- Navigate to the 'NotionElectron-master' folder
	- run `chmod +x Uninstall.sh && ./Uninstall.sh`
